Adhara Voice Control.

Adhara Voice Control é um programa de controle de voz. Com ele, você pode controlar determinadas tarefas do seu computador ou de qualquer dispositivo ligado a ele.

A versão Home Edition, permite ao usuário ter o controle de tarefas simples do Windows, como o Explorador de Arquivos, navegador Google Chrome, bem como algumas funcionalidades dos programes incluídos no pacote Office.

Projeto Hobby: Robô desenvolvido com arduino.

Este jogo desenvolvido em HTML, CSS e Java Script baseado em um grande clássico dos vídeo games antigos, consiste em destruir a maior quantidade de asteroides possíveis sem colidir com os mesmos, além é claro, contando com a possibilidade de aumento de velocidade.

O joguinho é um projeto independente de alguém que simplesmente quis colocar em prática os conceitos simples de HTML, CSS e Java Script, possíveis erros podem ser encontrados, contudo, foi um ótimo trabalho e claro, certamente você se divertirá muito! Bom jogo!

Almach Learning - Software de simulados e tarefas para escolas

Este software desenvolvido em C# consiste de simulados de língua inglesa contemplando as habilidades de Reading, Grammar e Listening. O programa usado no desenvolvimento das atividades foi o Almach Learning. O programa foi desenvolvido com recursos próprios pelo acadêmico do curso de Sistemas de Informação da Universidade Federal do Piauí, José de Sousa Magalhães, também professor de inglês da instituição aqui apresentada.

Arm Paint - Sofware de desafio de desenhos e imagens.

Software desenvolvido em C# com banco de dados SQL Server para desafio de imagens e palavras. Cada desafio caracteriza-se por um objeto, imagem, fruta, animal, dentre outras opções, e ao ser escolhida, o sistema dá 20 dicas em inglês para que o mesmo possa escolher a que julgar correta. O software contribui no desenvolvimento do senso de identificação visual, leitura e vocabulário em inglês.
